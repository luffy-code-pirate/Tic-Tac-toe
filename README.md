# Tic Tac Toe Game

A simple and modern Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

- Two-player game
- Player name input
- Score board
- Wins, losses, and draws tracking
- Current turn indicator
- Restart round button
- Reset score button
- Clean and smooth UI

## Files

- `index.html`
- `style.css`
- `script.js`
